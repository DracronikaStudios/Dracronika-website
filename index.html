Dracronika Studio

Bonjour, bienvenue sur le site de Dracronika Studio.
Ce site est le site interne de l'équipe de développeurs.
Vous pouvez télécharger les programmes Python pour les modifier. Tout d'abord assurez vous d'avoir installé GitHub et d'avoir un compte sur la plateforme, de préférence votre compte Google spécifique au studio.

Ce site sert aussi de référence pour les projets qui, via les cahiers des charges, indiquent quoi faire. Il sert aussi à faire un état des lieux des membres de l'équipe, des rôles de chacuns, etc... Ce site est donc un launcher mais aussi un glossaire ainsi qu'un état des lieux.
